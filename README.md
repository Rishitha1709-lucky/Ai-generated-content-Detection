Multi-Modal Deep Learning Framework for AI-Generated Content Detection

Detects whether content is AI-generated or human-generated across 4 modalities.
Modalities covered: Text · Image · Audio · Video
Result: 80%+ classification accuracy across multi-modal test sets
Why it matters: One of few student-level projects tackling all 4 modalities simultaneously
Tools: Python · Deep Learning · Computer Vision · NLP · TensorFlow · Audio Processing
How to run:
1. Open the notebook in Jupyter
2. Select input modality (text/image/audio/video)
3. Run detection pipeline — outputs AI-generated or Human-generated label
